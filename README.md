# Close Idle Tabs

> Press a button to close inactive tabs

<img src="https://github.com/AnNOtis/close-idle-tabs/raw/master/src/assets/logo-128x128.png" width="100" height="100" >

## Run

Development:

```sh
yarn install
yarn run start
```

Build:

```sh
yarn run build
```
